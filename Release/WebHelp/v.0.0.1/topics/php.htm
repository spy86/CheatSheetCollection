Handling Exceptions

To catch exceptions do

try {
    # do something
} catch(Exception $e) {
    print "$e\n";
}
 

As not all of PHP and its libraries do consistenly throw real exceptions you might want to convert runtime errors (that would otherwise terminate the code execution) into catchable exceptions:

set_error_handler(function($errno, $errstr, $errfile, $errline ){
    throw new ErrorException($errstr, $errno, 0, $errfile, $errline);
});
